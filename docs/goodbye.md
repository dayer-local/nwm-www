# Goodbye

*The New Welsh Mesh website is winding down*

---

Cheers for stopping by. The **New Welsh Mesh** website is closing, but this isn't goodbye to the mesh itself.

We're keeping the **Discord** and all of this **documentation** up for everyone to read, they're not going anywhere. From here on, the Discord is our one active home for the community, that and the mesh itself, out on the air.

To be clear: **we're still here.** The network's still running, we still exist, we're just stepping back from running a website. If you want to get involved, come and find us on the Discord.
